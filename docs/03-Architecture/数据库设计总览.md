# 数据库设计总览

## 基础信息

- **数据库：** MySQL — `jack_campus`
- **入口：** `init-db.sql`
- **迁移目录：** `migrations/`（57 个迁移文件）
- **执行脚本：** `scripts/run-incremental-migrations.js`

## 数据库规范

1. **逻辑删除** — 使用 `deleted_at DATETIME` 字段，非物理删除
2. **时间戳** — 所有表含 `created_at` + `updated_at`
3. **外键** — 关键关联使用 FOREIGN KEY 约束
4. **迁移命名** — `NNN_<动作>_<对象>.sql`，如 `047_treehole_tag_visibility.sql`
5. **种子数据** — 通过迁移文件插入（Tag、区域等）
