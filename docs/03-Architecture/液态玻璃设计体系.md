# XMUMDorm 液态玻璃 / 玻璃拟态设计迁移手册

本文档从 `XMUMDorm-2.0.0-LYZZ/frontend` 抽取所有与「液态玻璃」「玻璃拟态（Glassmorphism）」相关的 **CSS 设计方案**，包含：**应用场景**、**参数设计**、**可迁移代码片段**、**性能降级策略**及**源文件索引**。便于你在其它项目中复用或抽象为 Design Token。

---

## 1. 技术共识与迁移前提

### 1.1 核心手段

| 手段 | 典型用途 |
|------|-----------|
| `backdrop-filter` / `-webkit-backdrop-filter` | 对**背后内容**做模糊、饱和度等，形成「透过磨砂玻璃」观感。 |
| 半透明 `background: rgba(...)` | 与 backdrop 叠加，避免纯模糊发灰。 |
| `border: 1px solid rgba(255,255,255,0.x)` | 玻璃边缘高光。 |
| `box-shadow` 大扩散 | 悬浮层级与「厚重玻璃」感。 |
| 多层 `radial-gradient` / `linear-gradient` | 折射高光、品牌色液态底（见厦大蓝按钮）。 |
| `::after` 伪元素 | 顶部柔光、脉冲光晕层（不增加 DOM）。 |
| `filter: url(#svgFilter)`、`@keyframes` 移动 `background-position` | 「液态」动效，**不一定**依赖 backdrop。 |

### 1.2 必写前缀

WebKit 内核（iOS Safari、部分 Android WebView）需同时写：

```css
backdrop-filter: blur(16px) saturate(180%);
-webkit-backdrop-filter: blur(16px) saturate(180%);
```

### 1.3 性能与可访问性

本仓库多处采用：

- **触控端** `@media (pointer: coarse)` 或 `(hover: none) and (pointer: coarse)` 下将 `backdrop-filter` 设为 `none`，避免滚动时空白/掉帧。
- **避免对 `backdrop-filter` 本身做高频动画**（Diary 中注释：易导致频闪）；可改为只动画 `transform` / `opacity`。

## 2. 建议抽象的设计 Token（新项目可直接用 `:root`）

以下为根据本仓库统计的 **blur / saturate** 档位，可迁移为变量：

```css
:root {
  /* 模糊半径档位 */
  --glass-blur-xs: 2px;    /* 轻量文字条、弹幕 */
  --glass-blur-sm: 6px;
  --glass-blur-md: 8px;    /* 轻遮罩、次要卡片 */
  --glass-blur-base: 10px; /* 工具栏、次级面板（出现频率最高） */
  --glass-blur-lg: 14px;   /* 卡片、标签、骨架 pill */
  --glass-blur-xl: 16px;   /* 树洞 header pill、商户头 */
  --glass-blur-2xl: 18px;  /* 帖子详情卡、浮层头 */
  --glass-blur-3xl: 20px;  /* 顶栏、榜单主玻璃 */
  --glass-blur-4xl: 22px;  /* Tab 壳、日程 FAB、日记弹层 */
  --glass-blur-5xl: 25px;  /* 日记/日程「重毛玻璃」 */

  /* 饱和度（与 iOS 控制中心类似） */
  --glass-saturate-ios: 180%;
  --glass-saturate-soft: 160%;
  --glass-saturate-btn: 1.25; /* 与 180% 不同，用于按钮 saturate(1.25) */

  /* 常见表面色 */
  --glass-surface-white-92: rgba(255, 255, 255, 0.92);
  --glass-surface-white-72: rgba(255, 255, 255, 0.72);
  --glass-surface-white-55: rgba(255, 255, 255, 0.55);
  --glass-surface-white-45: rgba(255, 255, 255, 0.45);
  --glass-border-slate: rgba(226, 232, 240, 0.9);
  --glass-shadow-deep: 0 18px 60px rgba(2, 6, 23, 0.14);
}
```

## 3. 方案 A：全局导航玻璃（顶栏 + 底栏）

### 应用场景

- 页面顶部 **粘性顶栏**，滚动时内容从下方「透」过。
- 底部 **Tab 导航外壳**，浅灰玻璃避免发暗。

### 参数设计

| 元素 | 背景 | backdrop | 备注 |
|------|------|------------|------|
| 顶栏 | 白渐变 `rgba(255,255,255,0.84)` → `0.70` | `saturate(180%) blur(20px)` | 顺序：`saturate` 在前与仓库一致 |
| Tab 外壳 | `rgba(241, 245, 249, 0.92)` | `saturate(180%) blur(22px)` | 圆角约 `18px`，细灰边框 |

### CSS（迁移片段）

**顶栏** — 源：`src/components/TopBar.css`

```css
.top-bar {
  position: sticky;
  top: 0;
  z-index: 100;
  background: linear-gradient(
    to bottom,
    rgba(255, 255, 255, 0.84),
    rgba(255, 255, 255, 0.70)
  );
  backdrop-filter: saturate(180%) blur(20px);
  -webkit-backdrop-filter: saturate(180%) blur(20px);
  border-bottom: 1px solid rgba(255, 255, 255, 0.18);
}
```

**Tab 栏外壳** — 源：`src/components/TabBar.css`

## 6. 方案 D：日程页「重毛玻璃」+ 今日液态渐变（双技术）

### 应用场景

- **智能横幅**、**每日课程容器**、**FAB**、**弹层**：`blur(25px) saturate(180%)` 一类重玻璃。
- **今日高亮**：**不依赖 backdrop**，用 `linear-gradient` + `background-size` + `@keyframes scheduleLiquid` 制造流动色带；上层 `::after` 做 **shimmer** 扫光。

### 参数设计

| 区域 | backdrop | 背景 |
|------|----------|------|
| `.schedule-smart-banner` | 25px + saturate(180%) | `rgba(255,255,255,0.55)` |
| `.schedule-day` | 同上 | 半透白 |
| `.schedule-icon-btn` | 14px | 圆按钮 |
| `.schedule-fab` | 22px + saturate(180%) | `rgba(255,255,255,0.45)` |
| `.schedule-modal-overlay` | 8px | 暗遮罩 `rgba(2,6,23,0.45)` |
| `.schedule-modal` | 25px + saturate(180%) | 弹窗主体 |
| `.schedule-modal-head` | 22px | Sticky 头 |
| `.schedule-import-card` 等 | 8px | 较轻 |

### CSS（迁移片段）

**重玻璃日卡片 + 今日液态** — 源：`src/pages/Schedule.css`

```css
.schedule-day {

  background: rgba(255, 255, 255, 0.50);

  -webkit-backdrop-filter: blur(25px) saturate(180%);
  overflow: hidden;
  box-shadow: 0 16px 50px rgba(2, 6, 23, 0.10);

}

.schedule-day.schedule-today {
  background: linear-gradient(120deg, rgba(74, 222, 128, 0.95), rgba(59, 130, 246, 0.95), rgba(74, 222, 128, 0.95));

  border: 1px solid rgba(255, 255, 255, 0.40);
  box-shadow: 0 20px 40px rgba(59, 130, 246, 0.30);
}

.schedule-day.schedule-today::after {
  content: '';
  position: absolute;
  inset: 0;
  pointer-events: none;
  background: linear-gradient(120deg, rgba(255,255,255,0) 0%, rgba(255,255,255,0.22) 40%, rgba(255,255,255,0) 70%);
  transform: translateX(-140%);
  animation: scheduleShimmer 5s ease-in-out infinite;
  opacity: 0.85;
}

@keyframes scheduleLiquid {
  0% { background-position: 0% 50%; }
  50% { background-position: 100% 50%; }
  100% { background-position: 0% 50%; }
}

@keyframes scheduleShimmer {
  0% { transform: translateX(-140%); }
  60% { transform: translateX(-140%); }
  100% { transform: translateX(140%); }
}
```

## 10. 方案 H：广场 AboutUs — 非典型 backdrop + SVG 液态光晕

### 应用场景

- **扫光圣光**：`backdrop-filter: brightness(1.08)` 配合 `mix-blend-mode: screen`（对背后地图轻微提亮）。
- **弹幕条**：极轻 `blur(2px)` + `mask-image` 右侧淡出（注释「液态消失感」）。
- **贴图标签**：`blur(6px)`。
- **地图编辑浮层**：`blur(18px)`。
- **Freshman 贴纸**：`filter: url(#freshmanAura)`（SVG 滤镜，定义在 `AboutUs.jsx`），属 **液态光晕**，与 CSS backdrop 互补。

### 移动端降级（扫光）

源：`src/pages/AboutUs.css`

```css
@media (hover: none) and (pointer: coarse) {
  .circular-flare {
    backdrop-filter: none;
    -webkit-backdrop-filter: none;
    opacity: 0;
    animation-name: squareSunSweepMobile;
  }
}
```

### Liquid Aura 思路摘要（需配合 JSX 内 SVG）

源：`AboutUs.css` 约 `.square-sticker[data-id='freshman']::after`：

- 复制背景图到伪元素，`filter: url(#freshmanAura)`；
- `animation: squareFreshmanBreathe` 控制透明度与 `scale`；
- `::before` 做旋转椭圆环 `border-top-color` 金色 + `drop-shadow`。

迁移时需同时复制 **SVG `<filter id="freshmanAura">...</filter>`**（见 `src/pages/AboutUs.jsx` 内注释「SVG filters」）。

## 12. 迁移检查清单（给新项目）

1. **是否必须支持 iOS WebView**：是则始终加 `-webkit-backdrop-filter`。
2. **是否在滚动容器内叠多层 backdrop**：考虑树洞方案 — 触控端降级或减层。
3. **是否在低端机动画 backdrop**：参考 Diary — 改为动画 `transform`，或媒体查询关 blur。
4. **深色模式**：本手册摘录以浅色为主；迁移时建议为 `[data-theme="dark"]` 另配表面色与边框色。
5. **对比度**：玻璃上的文字需 `text-shadow` 或底渐变（见树洞 `.treehole-glass-text`）以保证 WCAG 感知对比。
