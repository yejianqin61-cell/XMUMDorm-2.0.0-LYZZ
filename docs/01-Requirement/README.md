## 核心需求模块

| 模块 | 需求概要 |
|------|----------|
| 认证系统 | 学号/邮箱 + 密码登录，学生/商家注册，JWT 鉴权 |
| 树洞/帖子 | 发帖、评论（二级）、点赞、Tag 分类、图片上传 |
| 食堂 | 区域→店铺→商品三级结构，点评系统、排行榜 |
| 广场 | 热搜榜、校园此刻、轮播、四宫格入口 |
| 社团 | 创建/加入/关注社团，社团动态 |
| 二手市场 | 商品发布、搜索、收藏 |
| 跑腿 | 任务发布、接单 |
| 一站通 | 课程评价、成绩查询等学校服务集成 |
| 等级系统 | 经验值、等级徽章、升级弹窗 |
| 管理员后台 | Dashboard、用户管理、举报审核、内容管理、公告 |
