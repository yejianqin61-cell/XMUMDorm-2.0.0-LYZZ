# Dorm 食堂模块入口页设计文档（V3.0 → V3.1）

**项目：** Dorm
**模块：** 食堂（Dining）
**版本：** V1.1
**开发阶段：**

- V3.0：核心功能开发
- V3.1：扩展能力开发

------

# 一、设计目标

食堂模块首页从地图式入口调整为工具型首页。

目标：

1. 帮助用户快速决定吃什么
2. 提高食堂模块打开频率
3. 降低操作成本
4. 为后续社交、推荐功能预留扩展

核心流程：

```
搜索
↓
发现内容
↓
查看热门
↓
进入对应食堂
↓
浏览菜品
```

------

# 三、V3.0 功能需求

------

## 1. 搜索模块

### 功能

统一搜索：

- 菜品
- 美食文章

支持模糊搜索。

示例：

用户输入：

```
榴莲
```

结果：

```
菜品：

榴莲千层
榴莲披萨

文章：

《XMUM榴莲攻略》
《榴莲爱好者测评》
```

------

### 后续扩展（V3.1）

支持：

```
麻辣
早餐
低卡
夜宵
甜食
```

标签搜索。

------

## 2. 推荐轮播模块

### 功能

首页顶部推荐内容展示。

内容组成：

### 内容推荐

示例：

```
今日热门菜品

新品上线

限时活动

推荐内容
```

### 商家广告

示例：

```
新品宣传

折扣活动

商家推广
```

原则：

```
内容推荐优先
广告作为补充
```

------

## 3. 食堂入口模块

### 功能

快速进入不同食堂区域。

Dorm 当前入口：

```
Bell

D6

LY3

B1

Others
```

说明：

| 入口   | 内容          |
| ------ | ------------- |
| Bell   | Bell食堂区域  |
| D6     | D6食堂区域    |
| LY3    | LY3食堂区域   |
| B1     | B1食堂区域    |
| Others | 其它商家/区域 |

点击：

```
进入对应区域页面
```

------

## 4. 排行榜模块

### 功能

展示热门内容。

Tab：

```
菜品榜

店铺榜

新品榜
```

数据：

```
由ClaudeCode读取当前排行榜已有榜单
```



------

## 5. 今天吃什么

### 功能

帮助犹豫用户快速选择。

展示：

```
不知道吃什么？

[帮我决定]
```

点击：

随机推荐内容。

------
