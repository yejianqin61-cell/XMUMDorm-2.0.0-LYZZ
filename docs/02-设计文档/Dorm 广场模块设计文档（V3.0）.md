# Dorm 广场模块设计文档（V3.0）

**项目：** Dorm
**模块：** 广场（Square）
**版本：** V2.0
**开发阶段：** V3.0
**设计方向：** 极简风 / 原生 App 风格 / 卡片式布局 / 内容优先

------

# 一、模块定位

广场页不承担具体业务功能，而作为 Dorm 各模块的信息聚合中心。

定位：

> “校园里正在发生什么”

核心职责：

- 展示校园热点
- 聚合核心入口
- 展示实时动态
- 为各模块导流

不承担：

- 社团内部通知中心
- 出物详情页
- 跑腿详情页
- 学校服务页面

这些内容进入对应模块处理。

# 三、页面模块

------

## 1. 热搜榜模块

### 功能定位

校园热点聚合入口。

参考：

- 虎扑话题榜
- 抖音热搜

目的：

- 提升广场活跃感
- 引导用户发现内容
- 运营校园话题

------

### 展示形式

```
🔥 热搜榜

1 榴莲节开始了 🔥

2 ACM开始招新 🔥

3 Bell新品怎么样 🔥

4 求高数学习搭子 🔥

5 宿舍停水通知 🔥

6 D6今天排队爆炸 🔥

7 今天学士服好多人 🔥
```

规则：

前三名：

```
#1 红色
#2 橙色
#3 黄色
```

其余：

```
灰色
```

显示：

```
标题

讨论数（可选）

趋势标识（可选）
```

例如：

```
榴莲节 🔥 123讨论
```

不使用：

```
670万热度
```

避免前期数据不真实。

------

### 创建方式（V3.0）

创建人：

```
管理员
```

创建内容：

```
标题

描述（可选）

开始时间

结束时间
```

示例：

```
标题：

考试周

描述：

分享复习技巧、吐槽、求助
```

------

### 点击逻辑

点击：

```
榴莲节
```

进入：

```
热搜专属讨论页
```

页面结构：

```
🔥 榴莲节

324讨论

描述内容

[参与讨论]

↓

帖子瀑布流
```

------

### 发帖逻辑

用户在热搜页发帖：

自动绑定：

```
当前热搜ID
```

帖子进入：

```
当前热搜讨论区
```

不与树洞互通。

规则：

```
树洞帖子
≠
热搜帖子
```
