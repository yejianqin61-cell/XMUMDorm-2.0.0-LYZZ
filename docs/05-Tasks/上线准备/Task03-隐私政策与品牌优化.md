## 一、任务拆解

### L03-Task001: 隐私政策页面（Google Play 上架必需）

| 项目 | 内容 |
|------|------|
| **复杂度** | ⭐ |
| **预估** | 0.25 天 |

**为什么需要**:
- Google Play 和 App Store 都强制要求隐私政策 URL
- 你的 App 收集了邮箱（注册）、图片（帖子/点评）、通知 token（Push）
- 必须有公开可访问的隐私政策页面

**两种方案**:

| 方案 | 做法 | 推荐 |
|------|------|:--:|
| **A: 站内页面** | `frontend/src/pages/PrivacyPolicy.jsx` → `/privacy` 路由 | ✅ |
| **B: 纯 HTML** | `frontend/public/privacy.html` → 静态页面，独立加载 | |

**推荐方案 A**，内容模板:

```
隐私政策

XMUMDorm（厦马小筑）重视你的隐私。

我们收集的信息：
- 邮箱地址（用于登录和验证码发送）
- 学号（用于身份验证）
- 你发布的内容（帖子、评论、点评、图片）

我们如何使用信息：
- 提供校园社交服务
- 发送通知（你关注的内容有更新）
- 不会出售或分享你的个人信息给第三方

数据安全：
- 密码经过加密存储
- 所有数据传输使用 HTTPS
- 你可以随时删除自己的内容

联系我们：yejianqin61@gmail.com

最后更新：2026-06-08
```

**验收标准**:
- [ ] 隐私政策页面可通过 URL 访问（如 `https://xmumdorm-200-lyzz-production.up.railway.app/privacy`）
- [ ] 内容覆盖了收集的所有数据类型
- [ ] 联系方式正确

### L03-Task003: 启动画面品牌优化

| 项目 | 内容 |
|------|------|
| **复杂度** | ⭐⭐ |
| **预估** | 0.5 天 |

**当前状态**: Capacitor 默认启动画面（白色背景 + 默认图标）已生成。
**目标**: 品牌化启动画面。

**两种方案**:

| 方案 | 做法 | 效果 |
|------|------|:--:|
| **A: 简单** | 更新 `capacitor.config.ts` 的背景色 + 图标 | 白色/浅蓝背景 + Logo 居中 |
| **B: 精致** | 设计师出图 → `resources/splash.png` → `npx capacitor-assets generate` | 品牌全屏启动图 |

**推荐方案 A**（快速）：

更新 `capacitor.config.ts`:
```ts
plugins: {
  CapacitorAssets: {
    iconBackgroundColor: '#E6F4FE',         // 浅蓝
    iconBackgroundColorDark: '#0a0a0f',     // 深色模式
    splashBackgroundColor: '#E6F4FE',       // 启动画面背景
    splashBackgroundColorDark: '#0a0a0f',
  },
},
```

如果你有设计好的启动图，放到 `resources/splash.png`（2732×2732），然后重新跑 `npx capacitor-assets generate`。

**验收标准**:
- [ ] 冷启动 App 时启动画面不出现"白屏闪烁"
- [ ] 图标和背景色与品牌一致
- [ ] 启动画面消失后无缝过渡到 App 首页
