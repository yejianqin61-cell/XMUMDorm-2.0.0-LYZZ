# 前端共享层任务清单

本目录将《前端共享层抽取开发计划》拆分为可逐项执行的任务文档。

执行原则：

- 按任务顺序推进
- 每完成一个任务，必须先验证再提交
- 每个任务单独使用一条 conventional commit
- 不跨任务混合提交

任务顺序：

1. `Task01-共享层基础设施搭建.md`
2. `Task02-抽取共享API层.md`
3. `Task03-抽取共享常量与纯工具层.md`
4. `Task04-抽取共享Query层.md`
5. `Task05-共享层验证与目录收口.md`
6. `Task06-第二批共享层预留任务.md`

说明：

- Task01 到 Task05 属于当前主线执行范围
- Task06 是第二批共享层的后续任务占位，当前不应提前实施
