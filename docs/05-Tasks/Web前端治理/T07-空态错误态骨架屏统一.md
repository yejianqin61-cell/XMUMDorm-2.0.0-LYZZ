# T07 空态错误态骨架屏统一

## 1. 任务目标

统一 `EmptyState`、`ErrorState`、`PageSkeleton` 的视觉和 props 结构，取代页面内零散状态块。

## 2. 影响范围

- `frontend/src/components/ui/EmptyState.jsx`
- `frontend/src/components/ui/ErrorState.jsx`
- `frontend/src/components/ui/PageSkeleton.jsx`
- 相关样式文件

## 3. 不可触碰边界

- 不批量重写所有页面状态逻辑
- 不做复杂插画系统

## 4. 实施步骤

1. 统一三者的 props 结构和样式语言。
2. 统一 CTA 区域、标题、描述、图标或占位视觉。
3. 为 `PageSkeleton` 细分列表、详情、卡片、表单骨架。
4. 选取代表性页面接入验证。

## 5. 验收方式

- 三类状态组件视觉统一
- 代表性页面可复用
- `npm run build:web` 通过

## 6. 推荐提交信息

`refactor(web-ui): unify empty error and skeleton states`

## 7. 前置依赖

- `T01-Token基础盘点与命名落地.md`
- `T02-Token兼容映射与样式接管.md`
- `T03-Card基础件统一.md`

## 8. 后置影响

- 为后续页面模板层统一 loading / empty / error 提供标准状态积木
