# T12 详情页模板落地

## 1. 任务目标

建立 `DetailPageLayout`，并用一个高频详情页完成模板验证，统一正文区、meta 区、评论区与右侧辅助区关系。

## 2. 影响范围

- `frontend/src/components/templates/DetailPageLayout.jsx`
- 一个代表性详情页
- 相关样式文件

## 3. 不可触碰边界

- 不同时迁移多个详情页
- 不把页面业务请求下沉进模板层

## 4. 实施步骤

1. 抽 `DetailPageLayout` 主骨架。
2. 设计 header、content、meta、comments、aside 插槽。
3. 接入 `PageHeader` 与现有基础件。
4. 选择 `PostDetail` 或 `FoodDetail` 做代表页验证。

## 5. 验收方式

- 详情页正文宽度受控
- 结构区分清晰
- 代表页已完成模板接入
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-template): add shared detail page layout`

## 7. 前置依赖

- `T09-页面头部与区块头模板统一.md`

## 8. 后置影响

- 为高频详情页迁移提供统一骨架
