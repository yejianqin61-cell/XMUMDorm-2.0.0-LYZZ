# T05 表单基础件统一

## 1. 任务目标

统一 `Input`、`Textarea`、`Select` 三类表单基础件，建立共享 field 样式规则。

## 2. 影响范围

- `frontend/src/components/ui/Input.jsx`
- `frontend/src/components/ui/Textarea.jsx`
- `frontend/src/components/ui/Select.jsx`
- 相关样式文件

## 3. 不可触碰边界

- 不做全站表单页重构
- 不改接口请求和提交逻辑
- 不把具体业务字段耦合进基础组件

## 4. 实施步骤

1. 统一 field 视觉规则：
   - height
   - border
   - radius
   - focus
   - error
2. 设计共享 props：
   - `label`
   - `hint`
   - `error`
   - `required`
   - `disabled`
   - `prefix`
   - `suffix`
3. 抽共享 field 样式。
4. 选取 1 到 2 个表单页面做代表性接入验证。

## 5. 验收方式

- `Input`、`Textarea`、`Select` 样式一致
- 错误态与禁用态统一
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-ui): add shared form field primitives`

## 7. 前置依赖

- `T01-Token基础盘点与命名落地.md`
- `T02-Token兼容映射与样式接管.md`
- `T04-Button基础件统一.md`

## 8. 后置影响

- 为发布页、编辑页、登录页提供统一表单积木
