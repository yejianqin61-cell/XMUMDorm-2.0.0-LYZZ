# T21 全局头部与左侧导航落地

## 1. 任务目标

在主壳已接线的前提下，落地桌面主站的全局头部和左侧导航，让主壳具备可识别的网页级主导航结构。

## 2. 影响范围

- `SiteHeader` 相关组件
- `SiteSidebar` 相关组件
- 主壳导航样式

## 3. 不可触碰边界

- 不在本任务中塞入复杂业务推荐块
- 不批量修改页面内部模板
- 不顺手统一后台导航

## 4. 实施步骤

1. 落地主站全局头部。
2. 落地左侧一级导航。
3. 保持导航项结构可复用、可扩展。
4. 完成后执行 `npm run build:web`。

## 5. 验收方式

- 全局头部与左侧导航已稳定挂入桌面壳
- 一级导航高亮与入口结构清晰
- 样式语言与现有视觉规范一致
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-shell): add site header and primary sidebar`

## 7. 前置依赖

- `T20-主站路由壳接入与Tab职责降级.md`

## 8. 后置影响

- 为首页、工作台、列表页的壳下验证提供真实导航环境
