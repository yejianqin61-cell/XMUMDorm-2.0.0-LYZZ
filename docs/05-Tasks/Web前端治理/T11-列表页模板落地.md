# T11 列表页模板落地

## 1. 任务目标

建立 `ListPageLayout`，并用一个高频列表页完成模板验证，形成“页面头部 + 筛选条 + 主列表 + 右侧辅助栏”的标准骨架。

## 2. 影响范围

- `frontend/src/components/templates/ListPageLayout.jsx`
- 一个代表性列表页
- 相关样式文件

## 3. 不可触碰边界

- 不同时改多个列表页
- 不在本任务中替换 Web 主壳

## 4. 实施步骤

1. 抽 `ListPageLayout` 主骨架。
2. 支持主列、可选右栏、页脚区域。
3. 接入 `PageHeader` 和 `FilterBar`。
4. 选择 `SquareTrendingList` 或 `MerchantList` 做代表页接入。

## 5. 验收方式

- 列表页主骨架可复用
- 代表页已完成模板接入
- 页面结构明显更像桌面网页
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-template): add shared list page layout`

## 7. 前置依赖

- `T09-页面头部与区块头模板统一.md`
- `T10-筛选条模板统一.md`

## 8. 后置影响

- 为后续高频列表页迁移提供标准骨架
