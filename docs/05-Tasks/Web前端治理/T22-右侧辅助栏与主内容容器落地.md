# T22 右侧辅助栏与主内容容器落地

## 1. 任务目标

补齐桌面壳的主内容容器与右侧辅助栏，使 Step 3 模板页能更自然地落在统一网页容器中。

## 2. 影响范围

- `ShellContent` / `SiteAside` 相关组件
- 主壳布局样式

## 3. 不可触碰边界

- 不强行把重业务模块塞进右侧栏
- 不在本任务中同时改造多个业务页面

## 4. 实施步骤

1. 落地主内容容器层。
2. 落地右侧辅助栏的最小承载结构。
3. 确保窄屏下右侧栏可以合理退化。
4. 完成后执行 `npm run build:web`。

## 5. 验收方式

- 主内容容器与右侧辅助栏结构已可用
- Step 3 模板层可在新壳中稳定承载
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-shell): add site aside and main viewport container`

## 7. 前置依赖

- `T21-全局头部与左侧导航落地.md`

## 8. 后置影响

- 为 `T24` 到 `T26` 的代表页壳下验证提供完整容器
