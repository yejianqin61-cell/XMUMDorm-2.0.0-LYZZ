# T13 表单页模板落地

## 1. 任务目标

建立 `FormPageLayout`，并用一个高频表单页完成模板验证，统一表单头部、说明区、表单分组区和底部操作条。

## 2. 影响范围

- `frontend/src/components/templates/FormPageLayout.jsx`
- 一个代表性表单页
- 相关样式文件

## 3. 不可触碰边界

- 不一次性重写所有发布页
- 不把业务表单校验逻辑写进模板层

## 4. 实施步骤

1. 抽 `FormPageLayout` 主骨架。
2. 设计 `header / notice / sections / actions / aside` 插槽。
3. 补 `StickyActionBar` 或等价操作区。
4. 选择 `PostNew` 或 `FoodCreate` 做代表页验证。

## 5. 验收方式

- 表单页结构统一
- 操作区位置稳定
- 代表页已完成模板接入
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-template): add shared form page layout`

## 7. 前置依赖

- `T09-页面头部与区块头模板统一.md`
- `T05-表单基础件统一.md`

## 8. 后置影响

- 为发布页、编辑页和管理表单页迁移提供统一骨架
