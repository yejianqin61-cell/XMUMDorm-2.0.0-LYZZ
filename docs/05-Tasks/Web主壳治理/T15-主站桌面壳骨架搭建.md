# T15-主站桌面壳骨架搭建

## 1. 任务目标

建立主站桌面 Web 壳的最小骨架结构，为后续主路由接入、导航落位和页面验证提供容器基础。

## 2. 影响范围

- `frontend/src/components/`
- `frontend/src/styles/`
- 新增主站壳相关组件

## 3. 不可触碰边界

- 不批量迁移业务页面
- 不改 `frontend-app/`
- 不改 `shared/*`
- 不在本任务中处理后台壳统一

## 4. 实施步骤

1. 新增主站 Web 壳骨架组件。
2. 建立 `Header / Sidebar / MainViewport / Aside` 占位结构。
3. 建立桌面壳基础样式与容器宽度规则。
4. 保证壳层只负责结构，不承载业务逻辑。

## 5. 验收方式

- 主站桌面壳组件已创建
- 三段式骨架结构清晰
- 样式与 Step 4 设计文档一致
- `npm run build:web` 通过
