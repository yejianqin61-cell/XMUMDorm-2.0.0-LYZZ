# T24 - 左侧主站导航 Twitter 化

**What to build:** 左侧主站导航改为 Twitter 式单层导航：每个模块以镂空线条图标和简洁文字呈现，消除无效说明与卡片堆叠，同时保持所有模块路由、当前页高亮、语言切换和管理员权限行为不变。

**Blocked by:** None - can start immediately.

**Status:** ready-for-agent

- [ ] 移除侧栏外层卡片、导航卡片、渐变圆点、模块说明与管理说明。
- [ ] 所有模块入口使用统一的线条图标与单行文字；管理员入口在底部以细分隔线区分。
- [ ] 激活项使用深色文字与左侧指示条；悬停仅使用浅灰背景，无阴影、上浮或渐变。
- [ ] 现有路由跳转、当前页高亮、管理员入口条件和中英文切换保持可用。
