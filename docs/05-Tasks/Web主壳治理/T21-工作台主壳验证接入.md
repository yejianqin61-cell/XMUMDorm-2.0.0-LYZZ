# T21-工作台主壳验证接入

## 1. 任务目标

以 `MyZone` 为代表页，验证 `DashboardPageLayout` 在新主壳中的落位和节奏是否成立。

## 2. 影响范围

- `frontend/src/pages/MyZone.jsx`
- 主站壳接入相关文件

## 3. 不可触碰边界

- 不在本任务中重构 `MyZone` 的业务模块
- 不混入其他个人中心页面批量迁移

## 4. 实施步骤

1. 让 `MyZone` 在新主壳下展示。
2. 验证工作台模板在主内容区的宽度、节奏和滚动表现。
3. 观察与左侧导航、右侧辅助栏的协作关系。

## 5. 验收方式

- `MyZone` 在新主壳下结构稳定
- 工作台节奏清晰
- `npm run build:web` 通过
