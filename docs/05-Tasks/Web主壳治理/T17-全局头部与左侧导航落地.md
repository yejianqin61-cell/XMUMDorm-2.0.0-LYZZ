# T17-全局头部与左侧导航落地

## 1. 任务目标

落地主站 Web 壳中的全局头部与左侧导航，明确一级模块导航和全局入口位置。

## 2. 影响范围

- 主站壳组件
- 全局导航相关样式
- Web 头部区域相关组件

## 3. 不可触碰边界

- 不在本任务中完成右侧辅助栏业务填充
- 不批量迁移各模块页面
- 不改后台导航

## 4. 实施步骤

1. 落地全局头部结构。
2. 落地左侧一级导航结构。
3. 接入当前主要模块的导航入口。
4. 统一桌面导航的视觉与交互状态。

## 5. 验收方式

- 头部结构完整
- 左侧导航可用
- 当前模块高亮正常
- `npm run build:web` 通过

## 6. 推荐提交信息
