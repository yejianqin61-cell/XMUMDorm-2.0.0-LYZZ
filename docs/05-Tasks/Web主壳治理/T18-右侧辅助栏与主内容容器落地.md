# T18-右侧辅助栏与主内容容器落地

## 1. 任务目标

落地主站桌面壳中的右侧辅助栏与主内容容器，让 Step 3 的页面模板有稳定落点。

## 2. 影响范围

- 主站壳组件
- 主内容容器样式
- 右侧辅助栏占位或轻量内容

## 3. 不可触碰边界

- 不在本任务中塞入重业务逻辑
- 不批量迁移各类列表页和详情页

## 4. 实施步骤

1. 建立主内容区容器宽度与滚动规则。
2. 建立右侧辅助栏占位结构。
3. 明确主区与辅区的响应式落位关系。
4. 保证 Step 3 模板可自然接入主内容区。

## 5. 验收方式

- 主内容区容器稳定
- 右侧辅助栏结构成立
- 桌面三段式节奏清晰
- `npm run build:web` 通过

## 6. 推荐提交信息

`feat(web-shell): add secondary aside and content viewport`
