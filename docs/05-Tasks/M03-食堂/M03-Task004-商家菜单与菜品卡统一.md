# M03-Task004 - 商家菜单与菜品卡统一

**What to build:** 学生进入商家后可通过现有分类导航定位菜品，并在统一的双列菜品图卡中比较图片、名称、商家、价格和评分。

**Blocked by:** M03-Task003 - 食堂分区商家列表收敛.

**Status:** ready-for-agent

- [ ] 保留商家菜单的分类导航和搜索能力，不把首页结构复制到菜单页。
- [ ] 菜品卡以固定比例图片为主要扫描锚点，信息层级统一。
- [ ] 价格、评分、商家名称和描述仅在数据存在时显示，描述最多两行。
- [ ] 点击菜品、图片预览和商家管理模式保持现有行为。
- [ ] `frontend` 生产构建通过，并验证分区、商家、菜品详情的跳转。
