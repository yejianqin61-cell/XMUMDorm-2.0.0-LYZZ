# M01-Task006-广场图标系统与发布入口收口

日期：2026-06-22  
优先级：P2  
状态：待开发

## 1. 任务目标

收敛广场页图标风格，避免彩色图标破坏整体感，同时重新处理发布入口与底部 Tab 的关系。

## 2. 改造范围

- 广场首页快捷入口图标
- 首页功能图标的颜色与风格
- 发布入口卡片/按钮
- 底部 Tab 协同观感

## 3. 改造要求

- 首页图标尽量统一风格
- 降低图标色彩杂讯，避免花绿感
- 发布入口要保留，但不能抢走主内容焦点
- 发布入口不能遮挡底部 Tab 栏
- Web 与 Capacitor 下都要保持安全区正确

## 4. 验收标准

- 图标系统风格更统一
- 发布入口更加克制
- 底部导航与首页内容不存在遮挡冲突

## 5. 测试要求

- 执行 `npm.cmd run build:web`
- 人工检查页面底部安全区与发布入口位置
