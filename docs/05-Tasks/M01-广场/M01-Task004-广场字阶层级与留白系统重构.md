# M01-Task004-广场字阶层级与留白系统重构

日期：2026-06-22  
优先级：P1  
状态：待开发

## 1. 任务目标

通过字阶对比、辅助信息弱化、模块间距和卡片内边距的统一，解决当前广场页“视觉层级混乱、页面呼吸感不足”的问题。

## 2. 改造范围

- `SquareHome.css`
- 广场首页所有标题、正文、辅助信息
- 相关首页组件中的层级文案结构

## 3. 改造要求

- 拉开一级标题、二级标题、正文、辅助信息的字号差
- 控制正文行高，避免长文本发闷
- 模块之间垂直间距统一提升到更舒展的节奏
- 卡片内部 padding 加宽，文字不要贴边
- 弱信息统一使用更低对比度的颜色

## 4. 验收标准

- 用户可以快速区分“页面主标题 / 模块标题 / 卡片标题 / 辅助信息”
- 模块与模块之间有明显呼吸感
- 文本排版不再拥挤

## 5. 测试要求

- 执行 `npm.cmd run build:web`
- 人工检查移动端窄屏下文本换行和行高是否自然
