# 编码规范

## 命名规范

| 类型 | 规范 | 示例 |
|------|------|------|
| 后端路由文件 | `<模块名>.js` 小写 | `routes/posts.js`, `routes/canteen.js` |
| 中间件文件 | `camelCase.js` | `middleware/adminAuth.js` |
| 数据库迁移 | `NNN_<描述>_<表名>.sql` | `migrations/047_treehole_tag_visibility.sql` |
| 前端页面 | `PascalCase.jsx` | `pages/Login.jsx`, `pages/TreeHole.jsx` |
| 前端组件 | `PascalCase.jsx` | `components/GlassView.jsx`, `components/EmptyState.jsx` |
| 前端 API 封装 | `api/<模块名>.js` | `api/auth.js`, `api/posts.js` |
| Context | `PascalCaseContext.jsx` | `context/AuthContext.jsx`, `context/ToastContext.jsx` |
| 样式文件 | `<同名>.css` | `pages/Login.css`, `components/Card.css` |

## API 设计规范

| 规范 | 说明 |
|------|------|
| RESTful | `GET /api/<resource>` 列表, `GET /api/<resource>/:id` 详情, `POST` 创建, `PATCH` 更新, `DELETE` 删除 |
| 鉴权 | 需登录接口用 `authenticateToken` 中间件 |
| 分页 | 列表接口支持 `?page=1&pageSize=20` |
| 错误响应 | `{ error: "message" }` + HTTP 状态码 |
| 图片上传 | Multer 单文件 `upload.single('image')` |

## 数据库规范

| 规范 | 说明 |
|------|------|
| 逻辑删除 | 使用 `deleted_at` 字段，非物理删除 |
| 时间戳 | 使用 `created_at` / `updated_at` |
| 索引 | 外键字段和常用查询字段加索引 |
| 表命名 | 小写 + 下划线：`post_likes`, `product_comments` |
