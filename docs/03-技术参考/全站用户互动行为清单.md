| 模块 | 触发行为 | 通知类型 |
|------|----------|----------|
| 树洞 | 点赞、评论、回复 | `like` / `comment` |
| 树洞 | 发布公告 | `announcement` |
| 二手 | 收藏 | `marketplace` |
| 二手 | 聊天消息 | `marketplace` |
| 一站通 | 点赞、评论、回复 | `handbook_comment` |
| 课程评价 | 评论、回复 | 通过 notifications 表 |

### 缺少通知（❌），已经在上面补充了

| 模块 | 缺失行为 | 影响 |
|------|----------|------|
| **食堂** | 点评、回复点评 | 用户收不到点评回复提醒 |
| **广场热搜** | 点赞、评论、回复 | 用户收不到互动提醒 |
| **校园此刻** | 点赞、评论、回复 | 用户收不到互动提醒 |
| **社团** | 关注、点赞、评论、回复 | 用户收不到社团动态提醒 |
| **跑腿** | 接单、完成 | 发单者收不到进度提醒 |
