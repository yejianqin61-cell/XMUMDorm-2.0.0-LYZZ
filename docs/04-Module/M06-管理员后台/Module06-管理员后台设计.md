# Dorm 管理员后台设计文档

版本：V3.0

------

# 一、模块概述

管理员后台是 Dorm 平台的运营管理中心。

管理员可通过：

我的 → 管理员中心

进入后台。

管理员后台主要负责：

- 用户管理
- 举报审核
- 内容管理
- 公告发布
- 数据统计
- 系统配置

------
