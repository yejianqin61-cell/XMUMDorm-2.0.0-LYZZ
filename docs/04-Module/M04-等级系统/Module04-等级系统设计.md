# Dorm 等级系统设计文档（V3.0）

## 1. 设计目标

等级系统的目标不是单纯“升级”，而是：

- 引导用户形成活跃行为
- 鼓励高质量内容贡献
- 提升用户留存
- 增强个人身份感与成就感
- 为后续勋章、活动、社交体系提供基础能力

---

# 2. 等级体系

采用：

- Lv1 ~ Lv6
- 越往后升级所需经验越高
- 前期升级快，给予用户正反馈
- 后期升级逐渐变慢

| 等级 |                    名称 | 累计经验 |
| ---- | ----------------------: | -------: |
| Lv1  |        Freshman（新生） |      100 |
| Lv2  |      Explorer（探索者） |      300 |
| Lv3  |   Contributor（贡献者） |      800 |
| Lv4  | Campus Star（校园达人） |     1800 |
| Lv5  |      Senior（资深成员） |     4000 |
| Lv6  |      Legend（校园传奇） |     8000 |

说明：

- 用户注册默认 Lv1
- 达到对应经验自动升级
- 升级时触发全局弹窗提醒

示例：

```text
🎉 恭喜升级 Lv4！

获得：

⭐ Campus Star 徽章
```

# 3. 经验获取规则

## 每日登录

| 行为      | 经验 | 每日上限 |
| --------- | ---: | -------: |
| 登录 Dorm |   +5 |        5 |

---

## 社区互动

### 点赞

| 行为              | 经验 | 每日上限 |
| ----------------- | ---: | -------: |
| 点赞树洞/热搜帖子 |   +1 |       15 |

说明：

- 最多获得15经验
- 防止无意义刷赞

---

### 评论

| 行为              | 经验 | 每日上限 |
| ----------------- | ---: | -------: |
| 评论树洞/热搜帖子 |   +5 |       15 |

规则：

- 评论字数至少 ≥5

---

### 发帖

| 行为         | 经验 | 每日上限 |
| ------------ | ---: | -------: |
| 发布树洞帖子 |  +10 |       15 |
| 发布热搜帖子 |  +10 |       15 |

规则：

- 内容长度 ≥10

---

## 食堂系统

### 食堂点评

| 行为         | 经验 | 每日上限 |
| ------------ | ---: | -------: |
| 普通点评     |  +10 |       30 |
| 优质点评奖励 |   +5 |       15 |

优质点评条件：

满足以下任意条件：

- 字数 ≥20
- 上传图片

示例：

有效：

```text
鸡腿饭一般般，肉有点柴，但是酱汁不错。
```

获得：

```text
+15经验
```

无效：

```text
好吃
```

获得：

```text
+10经验
```

---

# 4. 优质内容奖励机制

目的：

避免用户刷经验。

当帖子达到一定互动量：

| 条件         |    奖励 |
| ------------ | ------: |
| 帖子点赞 ≥10 | +20经验 |
| 帖子评论 ≥10 | +20经验 |

规则：

- 每篇帖子仅奖励一次
- 奖励自动发放

示例：

```text
🔥 你的帖子受到大家欢迎！

获得：

+20经验
```

# 5. 徽章系统

每个等级对应唯一徽章。

| 等级 | 徽章                |
| ---- | ------------------- |
| Lv1  | 🌱 Freshman Badge    |
| Lv2  | 🧭 Explorer Badge    |
| Lv3  | ✨ Contributor Badge |
| Lv4  | ⭐ Campus Star Badge |
| Lv5  | 🔥 Senior Badge      |
| Lv6  | 👑 Legend Badge      |

显示位置：

### 用户主页

```text
头像

Jack
👑 Lv6
```

---

### 帖子卡片

```text
Jack 👑 Lv6
2小时前
```

---

### 评论区

```text
Jack ⭐ Lv4
这个鸡腿饭真的不错
```

# 6. 升级流程

用户获得经验：

```text
执行行为
↓

增加经验值

↓

检测等级

↓

满足升级条件

↓

更新等级

↓

弹出升级动画

↓

授予徽章
```

# 7. 防刷机制

## 点赞防刷

规则：

- 重复点赞取消经验
- 同一帖子重复操作无效

1. 升级有反馈

3. 高质量内容收益更高
4. 防止刷经验
5. 徽章强化身份感
