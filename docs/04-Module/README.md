## 模块间依赖关系

```
M02 树洞（帖子系统）← 基础社交能力
  ├── M01 广场（热搜/校园此刻 → 引用帖子流模式）
  ├── M03 食堂（吃货广场 → 引用帖子流模式）
  └── M06 管理员后台（内容管理 → 引用帖子隐藏/删除）

M03 食堂 ← 独立模块
  └── 依赖 M02 树洞（点评/评论模式复用）

M04 等级系统 ← 全局横切
  └── 被 M01-M08 全部引用

M06 管理员后台 ← 全局横切
  └── 管理 M01-M08 全部内容
```
