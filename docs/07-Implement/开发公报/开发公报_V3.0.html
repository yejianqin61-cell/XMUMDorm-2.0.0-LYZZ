✅ 安全优势 (Strengths)

🛡️

全面参数化查询

所有 SQL 操作使用 pool.execute(sql, params)，SQL 注入完全防护

🧹

HTML 清理

sanitize-html 库应用于所有内容路由，XSS 存储型攻击防护

🔐

bcrypt 密码哈希

Salt rounds = 10，bcryptjs 库，防止密码泄露后破解

⛑️

Helmet 安全头

CSP / X-Content-Type / X-Frame / HSTS 等全套安全头

🚦

速率限制

全局 2500/15min + 认证 150/15min + 验证码 60s 冷却 / 日限 10 次

📋

全面审计日志

覆盖 14+ 动作类型，含 IP / UA / 角色，支持管理员查询

🔍

敏感词过滤

数据库驱动 + 内存缓存 (1min TTL) + 大小写不敏感匹配

🖼️

严格上传白名单

MIME + 扩展名双重校验，Sharp 服务端缩放，仅 image/* 类型

⚠️ 安全弱点 (Weaknesses)

🔑

硬编码 JWT 回退密钥

若未设 JWT_SECRET 环境变量，使用默认 'your-secret-key-change-in-production' 可被伪造

🗄️

无静态数据加密

邮件、用户内容等敏感数据以明文存储于 MySQL，数据库沦陷即全部泄露

🍪

Web 端 Token 存 localStorage

易受 XSS 攻击窃取，未使用 HttpOnly Cookie，无 refresh token 机制

📱

移动端 AsyncStorage 明文

Token + 用户数据在设备上非加密存储

🌐

CORS 全开放

app.use(cors()) 无 origin 限制，允许任意域名跨域请求

📏

无内容长度限制

帖子/评论内容无最大长度校验，存在存储滥用风险

🕵️

无 Schema 校验库

缺乏 Joi / Zod 等请求体结构校验，依赖 Ad-hoc 手写验证

🔄

JWT 不可刷新

7 天固定过期，无滑动会话或 refresh token

🔐 安全总评：B+ (78/100)

项目在 SQL 注入防护、XSS 防御、密码存储方面表现优秀，审计日志体系完善。 主要风险集中在 JWT 密钥管理（硬编码回退）、数据静态加密缺失、Web 端 Token 存储（localStorage XSS 风险）三个方面。 建议优先修复 JWT 回退密钥问题，评估引入 HttpOnly Cookie 方案，并添加请求 Schema 校验层。

🔏 用户信息隐私评估

基于 GDPR / 个人信息保护最佳实践的隐私合规审计

🟢 隐私维度合规度

📊 收集的个人数据类型

✅ 隐私保护优势

  • 邮箱域名限制（@xmu.edu.my）减少虚假注册
  • 课程评价支持匿名模式
  • 举报系统包含 "隐私侵犯" 类别
  • 验证码发送频率限制（60s 冷却 / 日 10 次）
  • 管理员操作全审计（含 IP 记录）
  • 封禁/禁言系统保护社区安全
  • 日记为私密内容（仅自己可见）

⚠️ 隐私风险

  • 无用户自助账号删除功能（仅管理员可软禁用）
  • 无数据导出 / 下载功能
  • 所有删除采用软删除（数据永久保留）
  • 无隐私政策 / 用户协议强制确认流程
  • 无 Cookie 同意横幅
  • IP 地址记录在审计日志中（无匿名化处理）
  • 学生 ID 与邮箱绑定存储
  • 无数据保留期限策略
  • 审计日志无防篡改保护

🔏 隐私总评：C+ (62/100)

项目在内容匿名化选项和社区安全保护方面有基本考量，但缺乏完整的隐私合规体系。 最紧迫的三项：① 实现用户自助账号删除和数据导出；② 添加隐私政策确认流程； ③ 建立数据保留和清理策略。对于面向高校学生的应用，建议在上线前至少完成这三项。

✨ 项目设计亮点

架构设计、工程实践、用户体验方面的创新与亮点

🏗️

Monorepo 优雅架构

三层分离：Express 后端、React Web 前端、React Native 移动端共享 API 契约。 61 个增量 SQL 迁移文件实现数据库版本控制，40+ 运维脚本支持平滑升级。

⚡

TanStack Query 状态管理

Web + Mobile 双端统一采用 TanStack Query 5，享受自动缓存失效、乐观更新、 后台重取、无限滚动分页等现代化数据管理能力。

🔐

多层安全防护

JWT 认证 + 数据库角色复核（防过期令牌提权）+ 敏感词过滤（内存缓存 1min TTL） + 封禁/禁言双轨处罚 + 全操作审计日志，构建纵深防御体系。

📱

跨平台功能对齐

Web 86 页面对齐 Mobile 69 屏幕。核心 17 模块全部双端覆盖，API 接口覆盖率 96.2%（152/158）， 确保双端用户体验一致性。

🎯

精致等级 & 经验系统

Lv1-Lv6 六级体系 + 8 种 EXP 获取规则 + 每日上限 + 质量奖励 + 互动里程碑。 不同等级徽章（emoji + 标签），升级弹窗动效，贯穿全平台展示。

🧪

测试驱动质量

移动端 416 测试用例全覆盖（20 模块 × 20 用例），后端 Supertest 集成测试 + 覆盖率阈值 CI。 测试辅助基础设施完善（Mock DB, Mock fetch, 6 个 Helper 文件）。

🖼️

智能图片处理管线

Multer 内存缓冲 + Sharp 自动缩放到 720px + WebP 转换 (q60) + S3/R2 对象存储 + 本地回退。 上传白名单严格控制（MIME + 扩展名），缩略图生成自动去除 EXIF。
