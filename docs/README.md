# 文档目录说明

本目录集中存放**非代码说明**与笔记，避免散落在仓库根目录。

| 路径 | 内容 |
|------|------|
| `docs/*.md` | 全站 API、UI、排行榜等评估与设计说明 |
| `docs/md笔记/` | 开发笔记、运行说明、Git 备忘等 |
| `docs/md/` | 功能与文件对照等短文 |
| `docs/战略/` | 产品与品牌相关长文 |
| `docs/notes/` | 从根目录迁入的随笔（如 `Dorm的品牌信用.md`） |
| `docs/frontend/` | 原 `frontend/docs/` 下的前端 UI 设计笔记 |

业务代码仍在仓库根目录的 `routes/`、`frontend/src/` 等位置；`html/` 为早期静态页，仅供参考。
