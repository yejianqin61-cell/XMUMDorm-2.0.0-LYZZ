# 文档目录说明

本目录集中存放**非代码说明**与笔记，按主题分 7 类。

| 路径 | 内容 |
|------|------|
| `01-项目评估/` | API 评估、UI 进度、版本分析、自检报告等 |
| `02-设计文档/` | 食堂 UI、帖子系统、液态玻璃、手册/市场等设计说明 |
| `03-技术参考/` | 功能与文件对照、代码学习顺序、手机预览指南 |
| `04-学习笔记/` | server.js、auth、database、React 入门等个人笔记 |
| `05-Git与运维/` | Git 命令备忘、回滚教程、上线注意事项 |
| `06-品牌与战略/` | 产品与品牌相关长文 |
| `07-开发计划/` | 按设计文档拆解的前后端开发计划与排期 |

业务代码仍在仓库根目录的 `routes/`、`frontend/src/` 等位置；`html/` 为早期静态页，仅供参考。
